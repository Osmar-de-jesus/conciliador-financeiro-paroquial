# ⛪ Conciliador Financeiro Paroquial

Sistema web desenvolvido para auxiliar paróquias na conciliação financeira entre relatórios de maquininhas de cartão e créditos recebidos na conta bancária.

---

## 📌 Objetivo

O sistema permite importar relatórios da operadora de cartão e extratos bancários, realizando uma comparação automática entre as transações registradas e os valores recebidos.

A ferramenta facilita a identificação de divergências, reduz erros manuais e auxilia no processo de conferência e prestação de contas financeira da paróquia.

---

## 🚀 Funcionalidades da Versão 1.0

- 📂 Importação de relatório da maquininha
- 🏦 Importação de extrato bancário
- 📄 Suporte aos formatos CSV, XLS e XLSX
- 🔍 Detecção automática das principais colunas
- ⚙️ Seleção manual de colunas quando necessário
- 💰 Comparação de valores entre maquininha e banco
- 📊 Exibição do resumo da conciliação
- ❌ Identificação de transações não conciliadas
- 📱 Interface responsiva para diferentes dispositivos

---

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
- SheetJS (XLSX)
